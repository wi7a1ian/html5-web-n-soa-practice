Instructions

Create Reader class in KO.Services namespace. Constructor of this class should accept one parameter which is selector for the control to read from (store it in a provate variable). It has one method - Read - which returns value from selected element.

Create Writer class in KO.Services namespace. Constructor of this class should accept one parameter which is selector for the dev to write to (store it in a provate variable). It has one method - Write - which writes passed text to selected element.

Create Processor class in KO namespace. Constructor of this class should accept selector for button it should attach to, as well as selectors for read and write elements. Processor class instantiate Reader and Writer objects which are used later. Processor has Process method which initiates read-write process.

Place each class in a separate external JS file.

Use revealing module pattern for your classes.
